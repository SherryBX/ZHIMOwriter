# ZHIMOwriter (智茉公众号排版工作台)

ZHIMOwriter 是一个专为微信公众号排版场景打造的 Markdown 编辑与预览工作台。它致力于解决“在本地写 Markdown，复制到公众号后台后排版、图片、框线丢失”的痛点，帮助创作者实现“所见即所得”的无缝排版体验。

![ZHIMOwriter Cover](./.github/assets/repo-cover.png)

> **左侧高效书写 Markdown，右侧实时预览公众号排版，一键无损复制到微信后台。**

---

## 🚀 项目亮点

- 🎨 **双主题模板一键切换**：内置 **Classic (经典暖橙)** 与 **BlackRed (黑红炫酷)** 两套高质感公众号排版主题，完美适配不同文风。
- 📝 **快捷插入与预设面板**：左侧提供可收缩的**快捷工具栏**与**预设面板**。支持一键插入作者、主编、往期推荐、精选评论等常用卡片，格式优雅低调，不抢正文风头。
- ⚙️ **个性化顶部标识 (Brand Label)**：支持自定义预览顶部的品牌标识（如默认的 `ZHIMO` ），更改后实时同步更新至微信复制模板和图片导出中。
- 💾 **永久本地保存 (Persistence)**：用户选择的主题、自定义的顶部标识、左侧修改的预设小工具配置，均通过 `localStorage` 自动永久保存在本地浏览器，刷新不丢失。
- 🖼️ **无限制图片支持 (Local & Remote Images)**：
  - 支持 **Windows 本地绝对路径**（例如：`E:/素材库/封面.png`）的直接预览渲染。
  - 支持图床远程链接（如 `https://`）。
  - **自动内联转码**：点击复制时，本地图片会自动在后台转码为 `base64` 嵌入 HTML，避免公众号后台因 `file://` 安全限制而无法加载图片。
- 📸 **Retina 级高清图片导出 (Zero-Flicker Screenshot)**：
  - 支持“复制为图片”功能，自动生成 2 倍视网膜清晰度（PNG）并写入剪贴板。
  - **零闪烁克隆技术**：使用离线 DOM 克隆渲染，截图时完全不影响当前页面滚动条，杜绝画面抖动与复位闪烁。
- 📎 **完美的格式保留 (WeChat Sanitizer Compliant)**：
  - **定制标题 & 引用**：对 `H1`、`H2`、`H3`、`Blockquote` 等做了公众号专属视觉优化，并适配了 Classic 主题的特色装饰横线。
  - **框线 & 分隔线修复**：使用 `section` 包装与微信安全策略兼容的行内 CSS 样式（如 `background-color` 画线法），解决复制后左右边框和下划线消失的通病。
  - **自动换行代码块**：代码块采用 `highlight.js` 高亮，并加入自适应换行，防止长代码在手机端被微信排版撑坏。

## 💻 本地开发指南

### 1. 依赖安装
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```
启动后会在本地开启 Vite 调试端口，使用浏览器访问即可。

### 3. 构建生产包
```bash
npm run build
```
打包后生成的静态资源会存放在 `dist` 目录中，支持无服务器静态托管。

## 📖 图片支持指引

编辑器内置了对本地路径的自动解析，你可以在 Markdown 中放心编写：

- **Windows 本地绝对路径**：
  `![示例图片](E:/图片库/配图/001.png)`
- **`file:///` 格式地址**：
  `![示例图片](file:///E:/图片库/配图/001.png)`
- **远程图床链接**：
  `![网络图片](https://yourcdn.com/images/001.png)`

> 💡 **复制提示**：在点击“复制到公众号”时，程序会自动在后台读取这些本地文件并转化为 `Base64` 编码；粘贴到公众号编辑器后，微信会自动将其解析为微信服务器的永久素材，无需手动重传图片。

## 🎨 技术栈

- **前端框架**：React 19 + TypeScript + Vite 6
- **Markdown 渲染**：markdown-it
- **代码高亮**：highlight.js
- **截图渲染**：html2canvas
- **页面设计**：Vanilla CSS 极致磨砂玻璃微动效（Glassmorphic UI Design）
